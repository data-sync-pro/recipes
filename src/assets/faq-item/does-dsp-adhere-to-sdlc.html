Yes. Data Sync Pro supports SDLC-aligned deployments through two options:

1. Manual JSON Export & Import (best for SMBs) Executables and Pipelines can be exported as JSON files and imported into another org. This straightforward approach works well for smaller customizations with a limited number of Pipelines and Executables.

2. Deployment with Cross-Org Data Sync via #Batch (best for enterprises) DSP can deploy any record-based configuration — including its own. The DSP deployment template can be downloaded from the Data Sync Pro website and imported into your org — from there, deployment is one click away.
