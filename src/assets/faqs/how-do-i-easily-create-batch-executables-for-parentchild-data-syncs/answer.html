Use the Migration Builder within the Pipeline to set up Batch Executables for syncing multiple related objects:

  1. Create a Pipeline record and set the Template Direction.
  2. Navigate to the Migration Builder tab.
  3. Add the root objects, then expand each one down to the leaf node you want to sync.
  4. Select the desired leaf node.
  5. In the modal that appears, choose the source and target matching fields, select the action, and click Create Executables to generate the required batch processes.
