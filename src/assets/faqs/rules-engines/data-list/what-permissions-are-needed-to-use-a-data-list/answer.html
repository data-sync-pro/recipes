To access and use a #Data List, users need the following:

  1. Permission Set — At minimum, the DSP: Data Sync Pro Starter permission set.
  2. Executable Record Access — At least Read access to the Executable record that defines the #Data List. By default, every #Data List is shared with all internal users through record sharing; this can be revoked by unchecking the field - All Internal Users Have Read Access.
  3. Custom Permission (when applicable) — If the Access Permissions field is set on the Executable, users must have the specified custom permission(s).
