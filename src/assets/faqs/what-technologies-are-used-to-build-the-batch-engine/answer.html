#Batch is built fully on Salesforce-native technologies—Apex for backend execution and Lightning Web Components (LWC) for the user interface. All processing stays entirely within your Salesforce org for maximum security. Unlike traditional Apex batch classes, DSP's #Batch runs on a proprietary optimized execution framework, purpose-built to handle large-scale data operations faster and more efficiently.

In fact, the entire DSP product is built on Apex and LWC.
